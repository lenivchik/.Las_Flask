О программе

Проверяются следующие разделы:

  • Индексная кривая (т.е. первая кривая) должна быть представлена глубиной, временем или индексом.
  • Для индексного канала допустимы только мнемоники DEPT, DEPTH, TIME или INDEX.
  • Время и дата могут быть включены в файлы LAS 2.0 при условии, что они выражены числом.
  • Внутренние пустые строки в любом месте раздела запрещены.
  • Раздел "~V" должен быть первым.
  • Раздел "~V" является обязательным.
  • Раздел "~W" ("WELL INFORMATION SECTION") является обязательным.
  • Раздел "~C" ("CURVE INFORMATION SECTION") является обязательным.
  • Раздел "~A" ("ASCII LOG DATA") является обязательным.
  • Раздел "~V" в файле LAS 2.0 может содержать только один раздел "~V".
  • Раздел ~V должен содержать строки: VERS, WRAP.
  • В файле LAS 2.0 может содержаться только один раздел "~W".
  • Раздел ~W должен содержать строки: "STRT", "STOP", "STEP", "NULL", "COMP", "WELL", "FLD", "LOC", "SRVC", "DATE".
  • В файле LAS 2.0 может содержаться только один раздел "~C".
  • В файле LAS 2.0 может содержаться только один раздел "~P".
  • В файле LAS 2.0 может содержаться только один раздел "~O".
  • Раздел данных ~A является последним разделом в файле.
  • Если индекс — глубина, единицы измерения должны быть M (метры) или F/FT (футы).
